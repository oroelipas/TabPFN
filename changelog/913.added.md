Add `auto` as a new `FEATURE_SUBSAMPLING_METHOD` option. When selected, it automatically uses `gini_feature_importance` (LightGBM-based) for datasets with more than 100k samples where feature subsampling is needed, and falls back to `balanced` otherwise. LightGBM is now a required dependency (previously optional via `pip install tabpfn[lightgbm]`).
