**Major release**: TabPFN-3 is now the default model. New users and existing users who do not pin a model will automatically get TabPFN-3 going forward. To use a previous model version, use the `create_default_for_version()` classmethod on `TabPFNClassifier` / `TabPFNRegressor`, or pass an explicit `model_path` to the estimator constructor to pin a specific model file.
